# 🚗 CO₂ Emissions – Linear Regression



This project is a machine learning study that analyzes and predicts CO₂ emissions of vehicles using linear regression. The aim was to estimate CO₂ emissions based on vehicle characteristics such as engine size, fuel consumption, and transmission type. The dataset contains 7,385 rows and 12 columns, covering various vehicle makes, classes, and fuel types. Both a simple and a multiple linear regression model were developed, compared, and evaluated using cross-validation and residual analysis.

## 📊 Model Results



|Model|Error Rate|
|-|-|
|Simple Linear Regression (engine size only)|\~12%|
|Multiple Linear Regression (engine size + fuel consumption)|\~8%|



The multiple linear regression model proved to be more efficient and was selected as the final model. It uses four features: engine size, city fuel consumption, highway fuel consumption, and combined fuel consumption. The prediction accuracy is approximately **92%**.
